## 📋 Page Templates Quick Guide

### Homepage Structure
```
1. Hero Section (gradient background)
   - Main heading (Display 2)
   - Subtitle (Lead text)
   - Primary CTA button
   - Secondary CTA button

2. Features Section
   - Section title (Display 3)
   - 4-column feature grid
   - Icons + titles + descriptions

3. Stats Section
   - 4 stat cards in a row
   - Numbers + labels + icons

4. CTA Section
   - Heading + description
   - Primary button
```

### Events Listing Structure
```
1. Filter Bar
   - Search input
   - Category dropdown
   - Date range picker
   - Price range slider
   - Clear filters button

2. Events Grid
   - 4 columns on desktop
   - Event cards with images
   - Pagination at bottom
```

### Event Detail Structure
```
1. Hero Section
   - Event image (left)
   - Title + description (right)
   - Book Now button
   - Share button

2. Tabbed Content
   - Details tab
   - Venue tab
   - Organizer tab
   - Reviews tab
```

## 🎯 Design Checklist

### Before Finalizing
```
✓ Colors match brand palette
✓ Typography follows hierarchy
✓ Spacing is consistent
✓ Components are reusable
✓ Responsive design considered
✓ Interactive states defined
✓ Accessibility checked
✓ Images optimized
✓ File exports ready
```

### Export Settings
```
Images:
• Format: PNG (graphics), JPG (photos)
• Quality: 80-90%
• Size: Multiple variants (1x, 2x)
• Naming: Descriptive, consistent

Icons:
• Format: SVG (scalable), PNG (compatibility)
• Size: 16px, 24px, 32px, 48px
• Color: Multiple variants
• Background: Transparent
```
