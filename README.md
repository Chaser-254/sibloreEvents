## Quick Start

### Prerequisites
- **Python 3.10+** - Latest Python version recommended
- **Git** - For version control
- **Code Editor** - VS Code, PyCharm, or similar
- **Modern Browser** - Chrome, Firefox, Safari, or Edge

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd siblore-events
   ```

2. **Create and activate virtual environment**
   ```bash
   python -m venv venv
   # On Windows:
   venv\Scripts\activate
   # On macOS/Linux:
   source venv/bin/activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Environment Setup**
   ```bash
   # Copy environment template
   cp .env.example .env
   # Edit .env with your settings
   ```

5. **Run migrations**
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```

6. **Create superuser**
   ```bash
   python manage.py createsuperuser
   ```

7. **Load initial data** (optional)
   ```bash
   python manage.py loaddata initial_data.json
   ```

8. **Run development server**
   ```bash
   python manage.py runserver
   ```

9. **Access the application**
   - **Main site:** http://127.0.0.1:8000/
   - **Admin panel:** http://127.0.0.1:8000/admin/
   - **API docs:** http://127.0.0.1:8000/api/docs/

```bash

ALLOWED_HOSTS=yourdomain.com,www.yourdomain.com

# Database Configuration
DATABASE_URL=postgresql://user:password@localhost/dbname

EMAIL_PORT=587
EMAIL_USE_TLS=True

MPESA_CONSUMER_KEY=your-mpesa-consumer-key
```

### Core Models
- **User:** Custom user model with role-based access control
- **Event:** Event information with pricing, capacity, and venue details
- **MerchandiseItem:** Product information with inventory management
- **Product:** Product information with affiliate marketing capabilities
- **ProductCategory:** Product categorization and organization

### Database Relationships
```
User (Custom Django User)
├── Profile (One-to-One)
│   ├── Profile Picture
│   ├── Bio
│   └── Business Name
├── Events (One-to-Many)
│   ├── Event Details
│   ├── Pricing Tiers
│   └── Venue Information
├── Merchandise (One-to-Many)
│   ├── Product Details
│   ├── Inventory
│   └── Pricing
├── Tickets (Many-to-Many)
│   ├── Purchase History
│   ├── Payment Status
│   └── QR Codes
└── Applications (One-to-Many)
    ├── Job Applications
    ├── Resume Uploads
    └── Application Status
```

## Deployment Guide

### Production Deployment

#### 1. Server Setup
```bash
# Update system packages
sudo apt update && sudo apt upgrade -y

# Install required packages
sudo apt install python3-pip python3-venv nginx postgresql postgresql-contrib
```

#### 2. Database Configuration
```bash
# Create database user and database
sudo -u postgres psql
CREATE DATABASE siblore_events;
CREATE USER siblore_user WITH PASSWORD 'secure_password';
GRANT ALL PRIVILEGES ON DATABASE siblore_events TO siblore_user;
\q
```

#### 3. Application Setup
```bash
# Clone repository
git clone <repository-url>
cd siblore-events

# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
pip install gunicorn psycopg2-binary

# Configure environment variables
cp .env.example .env
# Edit .env with production settings
```

#### 4. Gunicorn Configuration
```bash
# Create Gunicorn service file
sudo nano /etc/systemd/system/siblore-events.service
```

#### 5. Nginx Configuration
```bash
# Configure Nginx
sudo nano /etc/nginx/sites-available/siblore-events
sudo ln -s /etc/nginx/sites-available/siblore-events /etc/nginx/sites-enabled
sudo nginx -t
sudo systemctl restart nginx
```

#### 6. SSL Certificate
```bash
# Install SSL certificate
sudo apt install certbot python3-certbot-nginx
sudo certbot --nginx -d yourdomain.com -d www.yourdomain.com
```
